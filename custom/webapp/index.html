odML metadata SPARQL service

This server provides a searchable database of neuroscientific datasets published via both the CRCNS (Collaborative Research in Computational Neuroscience) and the data publication platform GIN hosted by the German Neuroinformatics Node.

odML (open metadata Markup Language) is a file format for storing heterogeneous metadata. The underlying data model offers an approach to store metadata in a structured human- and machine-readable way. Find out more about the format on the main odML project page.

odML can be exported to RDF to enable search multiple documents using the SPARQL query language.
